# Temperature Converter in Rust

A simple command-line tool to convert between Celsius and Fahrenheit.

## Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Input validation
- User-friendly interface

## Prerequisites

- Rust and Cargo installed on your machine

## Installation

1. Clone this repository
2. Navigate to the project directory:
3. And write cargo run in the terminal
   ```bash
   cd "Daily Life/Temperature Converters/TemperatureConverter-Rust-Slaytt"
   cargo run
   
## Example

Enter temperature: 25
Choose a conversion:
1. Fahrenheit to Celsius
2. Celsius to Fahrenheit
   2
   25°C is 77°F
