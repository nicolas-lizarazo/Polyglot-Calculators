# Description
This is a simple temperature converter, written in C++.
It's provide conversion from Celsius and Fahrenheit.
Using: <value> <unit>.

# How to use:
## Windows:
- Compile the program using a C++ compiler. For example clang++:
	```PowerShell
	clang++ main.cpp -o main.exe
	```
- Run the program:
	```PowerShell
	./main.exe <value> <unit>
	```
	 
## Linux:
- Compile the program using a C++ compiler. For example clang++:
	```bash
	clang++ main.cpp -o main
	```
- Run the program:
	```bash
	./main <value> <unit>
	```

## Example use:
```
./main 36.6 f
97.88
```
