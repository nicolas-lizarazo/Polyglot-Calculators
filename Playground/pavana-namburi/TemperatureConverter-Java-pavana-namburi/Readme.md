# Temperature Converter (Java)

A simple Java program to convert temperature between:
- Celsius
- Fahrenheit

## How it works
The program takes a temperature value and converts it using standard formulas.

## Conversion Formula
- Celsius to Fahrenheit: `(C × 9/5) + 32`
- Fahrenheit to Celsius: `(F − 32) × 5/9`

## How to run
1. Compile the program
```bash
    javac TemperatureConverter.java
```
2. Run the program
```bash
    java TemperatureConverter
```
